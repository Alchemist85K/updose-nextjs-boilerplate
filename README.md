# nextjs-boilerplate

A pre-configured Claude Code boilerplate for **TypeScript + Next.js + Tailwind CSS + Supabase** projects. Provides development rules, git conventions, and curated AI skills out of the box.

## Installation

```bash
npx updose add Alchemist85K/updose-nextjs-boilerplate
```

## What's Included

### Claude Code Configuration

- **CLAUDE.md** — Core development rules (task scoping, lint/format enforcement, documentation management)
- **Rules** — Git safety rules preventing accidental commits, pushes, and sensitive file exposure
- **Skills** — Conventional Commits workflow with pre-commit quality checks

### AI Skills

Pre-configured skill dependencies for common Next.js stack:

| Category | Skills |
|----------|--------|
| **Core** | Context7, TypeScript Advanced Types |
| **Next.js** | Best Practices, Cache Components, App Router Patterns |
| **React** | Composition Patterns, React Best Practices |
| **Styling** | Tailwind Design System, Web Design Guidelines, Frontend Design |
| **Backend** | Supabase + PostgreSQL Best Practices |
| **Auth** | Better-Auth Best Practices |

### Documentation Structure

The boilerplate sets up conventions for project documentation:

- `docs/plans/` — Feature plans with checklists
- `docs/review/` — Reviews, retrospectives, and lessons learned
- `docs/ARCHITECTURE.md` — System architecture
- `docs/DEVELOPMENT.md` — Development guide
